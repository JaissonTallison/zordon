# 🚀 ZORDON

Sistema de inteligência operacional para análise de dados empresariais.

## 🧠 Arquitetura

Monolito modular dividido em:

- zordon-api → regras e processamento
- zordon-data → banco de dados
- zordon-interface → frontend
- zordon-infra → infraestrutura

## ⚙️ Tecnologias

- Node.js
- PostgreSQL
- Docker
